Sprks Engine

Source Code

About

The Sprks Engine is a custom 2D game engine build on top of the MonoGame framework. Its codebase began with the Nullspace project back in 2023.

The engine also has a very barebones 3D rendering test that I plan on expanding on in the future. I haven't had the time to flesh-out the 3D pipeline but some of the back-end support is there. This includes things like Scene3D, Actor3D, Light3D, and a few basic 3D components.

I am very passionate about free and open source software, so the Sprks Engine is released on GitHub fully open source under the MIT license.

Features

  • Actor component system
  • Physics system
  • Asset management
  • Immediate-mode UI/layout framework
  • Awesome 2D lighting
  • Customizable rendering pipelines
    • Deferred 2D rendering
    • Experimental forward 3D rendering
  • Input and action binding systems
  • 2D tilemap creation

Projects

There are a handful of projects I've started developing in this engine. I have a few samples to show how to do some basic things in the engine. These can be found in the project repo, here.

Two games have been developed with this engine so far, Nullspace and SKIP_LOOP_PROCESS. Both of these games are unfinished as I've had to focused my attention on university coursework, but they are more fully-fledged projects built using this engine. They're not open source because I hope to publish them one day :]
