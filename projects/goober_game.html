Goober Game

About

Goober Game is a passion project that has spanned over a year's worth of work. It's a 2D space exploration survival game set in a small solar system comprised of a few planets of differing environments and climates. My scope may be massive but I have one hidden trick up my sleeve that's helped me get so much done... obsession and hyperfixation :D

The Tech

Goober Game has been a massive undertaking technically. The massive scope certainly doesn't help my case as a solo developer. This game runs in a custom engine designed by myself, which is detailed below.

The engine I've been building for this project is based on the MonoGame framework, and utilizes C# across its entire codebase, which is rapidly growing every day. This doesn't include things like JSON files for game config data importing/saving, and code in HLSL for shaders.
