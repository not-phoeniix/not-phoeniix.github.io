About

My Linux desktop is a project I've worked on over the course of a few years. The project consists of a handful of config files for various programs, a few handmade shell scripts for automation, and an interactive desktop UI that is built on top of an open source framework called Astal.

This project is the desktop UI that I use every single day on my personal computers. I've built it to be functional, efficient, and lightweight. It's changed over the years, swapping between frameworks, program combinations, and window managers.
